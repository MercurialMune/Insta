# Insta Merc!!

### a web app that allows users to post their photos and view and vote on other users' photos


## Specifications

| Behavior            | Input                         | Output                        |
| ------------------- | ----------------------------- | ----------------------------- |
| View posted photos | None | Display all images |
| View own images | none | Display owner's specific images|
| Vote and comment on images| click a button and input string text| leave a feedback on how one rates the posted images|

## Prerequisites

    - Python 3.6 required

## Set-up and Installation

    - Clone the Repository
    - Install python 3.6
    - Run "./start.sh" or "python3 manage.py server" from the terminal
